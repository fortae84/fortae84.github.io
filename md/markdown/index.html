Markdown HTML 변환하기

  1. NodeJS 설치

    • https://nodejs.org/ko

  2. 전역 markdown-it Package 설치

    • npm install -g markdown-it

  3. 변환 폴더 이동 후 명령어 실행

    • markdown-it "파일명".md -o index.html
